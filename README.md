# Shadow Fang: Rise of the Shadow Blade

A 3D platformer built with Three.js and React, featuring procedurally generated levels, multiple enemy types, boss battles, and a synth soundtrack.

**Play now:** https://profundi.eu/shadowfang

## Tech Stack

- Next.js 16 + React 19
- Three.js (React Three Fiber)
- TypeScript
- Tailwind CSS v4
- Web Audio API synth engine

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 to play.

### Build

```bash
npm run build        # Next.js static export
npm run build:single # Standalone single-file bundle via Vite
```

## Controls

- Arrow keys / WASD — Move
- Space — Jump (double jump available)
- X — Attack
- P — Pause
